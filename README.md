# GetEx API — Документація

## Перегляд документації

ReDoc потребує завантаження `openapi.yaml` через HTTP (не file://). Запустіть локальний сервер з папки API:

```bash
# Варіант 1 (Node.js)
npx --yes serve .

# Варіант 2 (Python 3)
python -m http.server 8080
```

Потім відкрийте в браузері: `http://localhost:3000` (serve) або `http://localhost:8080` (Python).

## Файли

| Файл | Призначення |
|------|-------------|
| `openapi.yaml` | Специфікація API (OpenAPI 3.0). Тут описуйте ендпоінти, схеми, приклади. |
| `index.html` | Сторінка з ReDoc для відображення документації. |
| `favicon.ico` | Іконка вкладки. |

## Редагування API

Усі зміни вносьте в `openapi.yaml`: нові `paths`, `components/schemas`, параметри, приклади відповідей. Після оновлення файлу перезавантажте сторінку в браузері.
